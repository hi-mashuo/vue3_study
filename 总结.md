# git 命令

### add

```
git add .
```

### commit

-m 和后面的 message 为提交信息

```
git commit -m "message"
```

### push

```
git push
```

### 查看分支

```
git branch
```

### 创建分支

name 为分支名字

```
git branch name
```

### 删除分支

慎用

```
git branch -d name
```

### 新建并进入分支

-b 为新建分支

```
git checkout -b name
```

### 提交分支

```
git push -u origin name
```

### 合并分支

记住进入要合并的分支

```
git merge name
```
